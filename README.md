# Exercícios DOM

Repositório com exercícios de **JavaScript** focados em **manipulação do DOM (Document Object Model)**, utilizados para treinar e consolidar conceitos de front-end.

Os arquivos demonstram, na prática, como:

- Selecionar elementos HTML via JavaScript  
- Manipular conteúdo e estilos dinamicamente  
- Tratar eventos (cliques, inputs, etc.)  
- Atualizar a página de forma interativa sem recarregar

## Objetivo

Servir como material de estudo e prática pessoal em:

- HTML  
- CSS  
- JavaScript  
- Manipulação do DOM  

Ideal para revisar conceitos básicos de front-end e reforçar lógica aplicada na interface.

## Estrutura do projeto

Cada exercício está organizado em pastas/arquivos contendo:

- `index.html` – estrutura da página  
- `style.css` (quando aplicável) – estilização básica  
- `script.js` – lógica em JavaScript responsável pela interação com o DOM  

> A organização pode variar um pouco entre os exercícios, mas a ideia central é sempre demonstrar a interação entre HTML, CSS e JavaScript.
